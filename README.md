# Texas-Salary-Analysis-and-Prediction
Analysis and prediction of salary trends in Texas using Python to support compensation and workforce decision-making.

## Business Objective
The objective of this project is to analyze salary patterns in Texas and identify key factors influencing compensation levels. The insights support data-driven workforce planning, compensation benchmarking, and salary prediction.

## Problem Statement
Understanding salary trends is essential for organizations to remain competitive, manage compensation costs, and attract talent. This project analyzes salary-related data to uncover drivers of compensation and build predictive insights.

## Dataset Overview
- Salary and workforce-related attributes
- Target variable: Salary
- Features include job role, experience, location, and related factors

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook

## Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature-level salary trend analysis
- Predictive modeling for salary estimation
- Business interpretation of results

## Key Insights
- Experience level is a major driver of salary variation
- Job role and location significantly impact compensation
- Salary distributions show clear stratification across roles

## Business Use Cases
- Compensation benchmarking
- Salary planning and budgeting
- Workforce and hiring strategy support

## Project Artifacts
- Python Notebook: `python/Texas Salary.ipynb`
- Final Report: `reports/Texas_Salary_Analysis_and_Prediction_Final_Report.pdf`

## Conclusion
This project demonstrates how data analysis and predictive insights can support compensation-related decision-making and workforce planning.
