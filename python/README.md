# Python Analysis

This folder contains the Python (Jupyter Notebook) used for analyzing salary data related to Texas employees.

### Key tasks performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Salary distribution and trend analysis
- Comparison across roles, departments, and experience levels

### Tools used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
